# Docker Elevator

Privilege Escalation by Docker

## Prerequisites

* Docker (daemon running by root)
* Compiler (gcc)
* User in docker group
* Internet connection

## Usage

```bash
chmod +x elevator.sh && ./elevator.sh
```

After finished, execute `/tmp/.supersh`
